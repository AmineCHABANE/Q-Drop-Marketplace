# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 7.x.x   | :white_check_mark: |
| < 7.0   | :x:                |

## Reporting a Vulnerability

Q-Drop Labs takes the security of its quantum temporal architecture seriously. 

If you believe you have found a security vulnerability in the CHRONOS-7 kernel or the OMEGA-9 neural fabric, please report it to our security team directly via our encrypted channels. 

**Do not report security vulnerabilities through public GitHub issues.**

We will acknowledge your email within 24 hours, and send a more detailed response within 48 hours indicating the next steps in handling your report.
